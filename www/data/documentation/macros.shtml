Macros for usage in event messages and alarms

%%Insert % character
%nName of event source object
%aIP address of event source object
%iEvent source object unique ID
%tEvent's timestamp is a form day-month-year hour:minute:second
%TEvent's timestamp as a number of seconds since epoch (as returned by time() function) - available from version 0.1.12
%cEvent code
%sEvent's severity code as number. Possible values are:
0= NORMAL
1= WARNING
2= MINOR
3= MAJOR
4= CRITICAL
%SEvent's severity code as text
%vNetXMS server version
%mEvent's message text (meaningless in event template)
%1 - %99Event's parameter number 1..99

Special characters

\tTab character
\nCR/LF character pair
\\Backslash character

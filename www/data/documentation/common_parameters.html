Appendix A (Description of Process.XXX parameters)

 

All Process.XXX parameters except Process.Count are used to obtain information about specific process(es) and have identical arguments. Common parameter format is

 

Process.XXX($1,$2)

 

where XXX is an actual process parameter, for example, it’s VM size.

 

Parameter arguments have the following meaning:

$1

Name of the process. If there is more than one process, result depends on second parameter.

$2

Modifier of parameter behavior when more than one process with given name exists. Can be the following:

min

minimal value among all processes named $1

max

maximal value among all processes named $1

avg

average value for all processes named $1

sum

sum of values for all processes named $1

If only one process with given name exists, this argument has no effect. If this argument is omitted, default value of “sum” is used.
